Specwizard
==========

`Specwizard <https://github.com/specwizard/specwizard>`__ (`Theuns et al 1998
<https://ui.adsabs.harvard.edu/abs/1998MNRAS.301..478T>`__,  `Schaye et al. 2003
<https://ui.adsabs.harvard.edu/abs/2003ApJ...596..768S>`__)
is a python package to compute and analyze mock quasar absorption spectra from cosmological simulations.

Examples can be found `within the project repository <https://github.com/specwizard/specwizard>`__
